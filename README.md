# AI-Based Deforestation Detection using CNN

## Overview
This project uses a Convolutional Neural Network (CNN) with TensorFlow to detect deforestation from satellite images.
It supports sustainability by identifying affected forest areas from image data.

## Dataset
Dataset used: [Amazon Rainforest Satellite Dataset (Kaggle)](https://www.kaggle.com/datasets/piyushkumar18/amazon-rainforest-satellite-data)

## Technologies
- Python
- TensorFlow / Keras
- Matplotlib
- NumPy
- Pandas

## Steps to Run
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Replace the dataset path in the code.
4. Run:
   ```bash
   python model_training.py
   ```
